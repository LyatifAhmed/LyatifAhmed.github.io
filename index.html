About Me

Hi! I’m Lyatif Redzheb, a Junior Data Engineer with a strong passion for creating efficient and scalable data pipelines. My technical expertise spans cloud platforms like AWS, programming in Python and SQL, and using big data tools such as Apache Spark and Kafka. I am dedicated to solving complex data challenges and improving data systems to help businesses gain actionable insights.

I completed the AiCore Data Engineering Bootcamp, where I honed my skills in building end-to-end data pipelines, automating workflows, and deploying machine learning models. One of my key projects involved developing an AWS-based data pipeline, which processed over 30k rows of data daily and improved processing times by 20%.

Currently, I’m a volunteer software developer for the Kind Giving Platform, where I collaborate with a team to enhance platform functionality and improve user experiences. This role has helped me apply my data engineering skills in a real-world setting while contributing to a meaningful cause.

What I Bring to the Table:

  • Experience with ETL/ELT processes, real-time data streaming, and large-scale data architectures
  • Strong problem-solving skills and a commitment to continuous learning
  • A collaborative mindset, having worked in agile teams to deliver impactful solutions

I’m always eager to learn and explore the latest advancements in cloud computing, data engineering, and machine learning. I’m excited about opportunities to leverage data to drive innovation and efficiency.
